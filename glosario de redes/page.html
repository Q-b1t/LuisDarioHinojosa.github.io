• Broadcast: Enviar mensaje a todos los puertos mediante un switch. Duando en el header dice que el destinatario con varios puertos del medio.
  • DHCP: Dinamic Host Configuration Protocol: Usa un administrador de red para determinar tu direccion de red y completar una ip para asignartela a tu computadora.
  • DSN: Domain Name Server
  • Media Access Control: Mac Address
  • ISO: International Standards Organization
  • IP: Internet protocol: 4bytes separados por un punto para identificar red.
  • ISP: Internet Service Provider.
  • OSI: Open Systems Interconnection Model
  • PDU: Protocol Data Unit
